# Earth Textures

Download these textures from Solar System Scope (CC-BY-4.0 License):
https://www.solarsystemscope.com/textures/

## Required Files

1. **earth_daymap_2k.jpg** - Earth day color map (Blue Marble)
   Download "Earth - Day Map" (2K resolution)

2. **earth_nightmap_2k.jpg** - City lights / night side
   Download "Earth - Night Map" (2K resolution)

3. **earth_clouds_2k.jpg** - Cloud layer
   Download "Earth - Clouds" (2K resolution)

4. **earth_normal_2k.jpg** - Surface normal/bump map
   Download "Earth - Normal Map" (2K resolution)

5. **earth_specular_2k.jpg** - Ocean specular highlights
   Download "Earth - Specular Map" (2K resolution)

## Alternative Source

NASA Visible Earth - Blue Marble:
https://visibleearth.nasa.gov/collection/1484/blue-marble

Place all files in this directory (frontend/public/textures/).
The app will fall back to a procedural Earth if textures are missing.
